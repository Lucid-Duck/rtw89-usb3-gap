# D-Link DWA-X1850 Rev B1

## Identity

| Field | Value |
|---|---|
| Marketing name | D-Link DWA-X1850, Rev B1 |
| VID:PID | `2001:332c` |
| Chip family | RTL8852AU family (binds to `rtw89_8852au_git`) |
| Generation | AX (Wi-Fi 6) |
| Firmware | `rtw8852a_fw-1.bin`, version 0.13.36.2 (52acc807) |
| Chip info from firmware | CID: 0, CV: 2, AID: 0, ACV: 0, RFE: 1 |

## Distinguishing features from the A1

The B1 is a genuinely different hardware revision from the A1. Observed differences:

1. **USB 2 descriptor string**: B1 reports "802.11ax WLAN Adapter" on the USB 2 side. A1 reports "802.11ac WLAN Adapter" on the USB 2 side (both switch to "802.11ax" after re-enumeration on USB 3).
2. **Installer shim**: B1 does NOT expose a `0bda:1a2b "DISK"` virtual CD-ROM before presenting the wireless device. The A1 does.
3. **Switch-mode register value**: B1 writes `0xb0f00` to register 0xc4. A1 writes `0xb0f10`. One-bit difference in the value.

## Identification pass results

### Hub topology (J5 Create powered USB 3.0 hub, 2026-04-11)

- Initial enumeration path: `3-1.3` (USB 2 high-speed via hub)
- Final path: `2-1.3` (USB 3 SuperSpeed via hub)
- Negotiated speed: **5000 Mbit/s SuperSpeed**
- Delay before re-enumerate: approximately 5 seconds
- Full dmesg: `evidence/dmesg/framework-morrownr-git/dwa-x1850-b1-hub-plug1-2026-04-11.log`

### Direct slot topology (Framework bottom-right USB-A, 2026-04-11)

- Initial enumeration path: `3-3` (USB 2 high-speed companion root)
- Final path: `2-2` (USB 3 SuperSpeed root)
- Negotiated speed: **5000 Mbit/s SuperSpeed**
- Delay before re-enumerate: approximately 5 seconds
- Full dmesg: `evidence/dmesg/framework-morrownr-git/dwa-x1850-b1-direct-plug1-2026-04-11.log`

## Topology invariance

Both hub and direct-slot runs produced bit-identical switch-mode signatures: same register value, same retry count, same delay. The switch-mode code is not sensitive to USB topology.
